Position relative

  1. Una capa arriba de los elementos que tienen posicionamiento
  2. Los demas respetan su lugar
  3. Se mueve en relacion a su posición original
  4. El display del elemento se respeta (block)
Span

Position Absolute

  1. Una capa arriba de los elementos que tienen posicionamniento estatico
  2. Los demás no respetan su espacio
  3. Se mueve con relación al contenedor padre que tenga un posicionamiento diferente a static
  4. Adapta el tamaño al contenido

Position Fixed

  1. Se queda fija en el viewport con su posicion original
  2. Una capa arriba de los elementos que tienen posicionamiento static
  3. Los demas no respetan su espacio
  4. Se mueve con relación al viewport
  5. Adapta el tamaño al contenido
